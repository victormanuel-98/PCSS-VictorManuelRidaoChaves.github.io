# Portfolio Interactivo

Proyecto visual en HTML/CSS/JS, sin backend ni base de datos.

## Versiones

- `cv/`: version con Vite para desarrollo y build.
- Raiz (`index.html` + `src/`): version estatica lista para desplegar.

## Desarrollo (recomendado)

```bash
cd cv
npm install
npm run dev
```

## Build

```bash
cd cv
npm run build
npm run preview
```
